# 🤖 Infrastructure as Code: Automated Deployment

Этот проект демонстрирует автоматическое развертывание (Deployment) веб-приложения на чистый сервер с использованием **Ansible**.
Плейбук настраивает окружение с нуля, устанавливает зависимости и запускает Docker-контейнеры.

---

## 🛠 Возможности
Плейбук `deploy.yml` выполняет следующие задачи:
1.  📦 **Установка зависимостей:** Обновляет `apt` кэш, устанавливает `git`, `docker.io` и `docker-compose`.
2.  🔄 **Клонирование кода:** Скачивает актуальную версию приложения из Git-репозитория.
3.  🚀 **Оркестрация:** Запускает приложение через `docker-compose`, пересобирая образы при необходимости.
