# Автоматизация развертывания с Ansible

Проект содержит Ansible Playbook для автоматической настройки сервера и развертывания контейнеризированного веб-приложения. Сценарий выполняет полную настройку окружения "с нуля" (Infrastructure as Code).

## Описание работы

Плейбук `deploy.yml` выполняет следующие действия на целевом хосте:

1.  **Подготовка окружения:** Обновление кэша пакетов `apt`.
2.  **Установка зависимостей:** Установка Docker, Docker Compose и Git.
3.  **Доставка кода:** Клонирование (или обновление) репозитория приложения в директорию `/opt/my-app`.
4.  **Развертывание:** Запуск `docker-compose up` для сборки и старта контейнеров.

## Структура проекта

*   `deploy.yml` — Основной сценарий автоматизации.
*   `hosts.ini` — Файл инвентаря, описывающий целевые сервера для деплоя.
